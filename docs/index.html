simpleSnackbar.js

A tiny and accessible toast/snackbar ES6 JavaScript plugin.

npm install simplesnackbarjs

or

Download

Getting Started

Adding simpleSnackbar.js to your existing project is very easy, just include the compiled CSS and JS files to your HTML document and initialise a snackbar.

You can also import the JavaScript if you are using a module bundler such as Webpack.

<!doctype html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, user-scalable=no, initial-scale=1.0, maximum-scale=1.0, minimum-scale=1.0">
        <meta http-equiv="X-UA-Compatible" content="ie=edge">
        <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/simplesnackbarjs@1.0.0/dist/simpleSnackbar.min.css">
    </head>
    <body></body>
    <script src="https://cdn.jsdelivr.net/npm/simplesnackbarjs@1.0.0/dist/simpleSnackbar.min.js"></script>
    <script>
        const snackbar = new simpleSnackbar('This is a message!').show();
    </script>
</html>

Options

Name Type Default Description Example
autohide Boolean true Determines whether the snackbar will automatically hide after being shown for the set duration.
close Boolean true Determines whether a close button is shown on the snackbar which the user can click to hide it.
icon Boolean true Determines whether an icon is placed on the snackbar for a clearer UI.
icons Object { success: '', info: '', warning: '', danger: '', } Defines the SVGs for the icons, out of the box we use Font Awesome icons.
transitionSpeed Number 250 Defines the speed of the transition in milliseconds
type String 'default' Defines the type of snackbar to initialise. Available types are 'default', 'success', 'info', 'warning', 'danger'

Methods

All methods return the instance of the snackbar you are calling to enable method chaining if needed.

Method Description
dispose This method removes the snackbar from the DOM when called.
hide This method hides the snackbar by moving it out of the viewport when called.
show This method moves the snackbar into the viewport when called.
toggle This method toggles between the hide and show states.

Events

If you want to hook in your own logic for your snackbar, you can use the list of events below.

Event Description
disposed This event is fired immediately when the snackbar has been removed from the DOM.
hide This event is fired immediately when the hide instance method has been called.
hidden This event is fired when the snackbar has been hidden from the user.
show This event is fired immediately when the show instance method has been called.
shown This event is fired when the snackbar has been made visible to the user.
