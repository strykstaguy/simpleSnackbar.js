simpleSnackbar.js

A tiny and accessible toast/snackbar ES6 JavaScript plugin.

npm install simplesnackbarjs

Download
Demo

Getting Started

Adding simpleSnackbar.js to your existing project is very easy, just include the compiled CSS and JS files to your HTML document and initialise a snackbar.

You can also import the JavaScript if you are using a module bundler such as Webpack.

<!doctype html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, user-scalable=no, initial-scale=1.0, maximum-scale=1.0, minimum-scale=1.0">
        <meta http-equiv="X-UA-Compatible" content="ie=edge">
        <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/simplesnackbarjs@1.0.0/dist/simpleSnackbar.min.css">
    </head>
    <body>
    </body>
    <script src="https://cdn.jsdelivr.net/npm/simplesnackbarjs@1.0.0/dist/simpleSnackbar.min.js"></script>
    <script>
        const snackbar = new simpleSnackbar('This is a message!').show();
    </script>
</html>
